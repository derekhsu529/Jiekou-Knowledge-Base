# 自动充值

启用 **“自动充值”** 后，当账户余额低于指定阈值时，系统将自动为您的账户充值。请按照以下步骤设置此功能：

### 1. 添加支付方式。

在控制台中访问 [支付方式](https://jiekou.ai/billing/payment-methods)。

如果您还没有支付方式，请点击 **“添加支付方式”**。您将被重定向到 Stripe 的安全页面以添加支付方式，我们不会存储任何支付信息。

### 2. 启用自动充值并配置设置。

在 **“自动支付”** 面板中点击 **“修改”** 以配置您的自动充值设置。

启用 **“启用自动充值”** 开关后，设置 **“当余额低于”** 和 **“将余额充值至”** 的有效值，然后点击 **“保存设置”** 以完成该过程。
