# Shell 模式

“Shell 模式” 是一种 TUI（Terminal User Interface）交互方式，让你在浏览器里直接体验 Claude Code 的原生终端工作流：进入 Shell 标签后会默认进入 Claude Code 的 TUI 界面，即可用对话的方式完成分析、改代码、运行命令与使用 Git 等任务。

## 进入 Shell 模式

点击右上角的 **Shell** 标签，即可进入 Shell 模式，默认连接到当前项目的云端沙箱环境上的 Claude Code 终端。连接就绪后，你会看到类似 Claude Code CLI 的终端交互界面。具体使用方式可以直接参考 [Claude Code 官方文档 - Interactive Mode](https://code.claude.com/docs/en/interactive-mode) 。
