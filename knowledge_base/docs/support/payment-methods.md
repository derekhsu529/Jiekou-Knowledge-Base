# 支付方式

> 支持的支付方式说明

JieKou.AI 使用 **Stripe** 进行所有支付处理。

您的所有支付信息都由 Stripe 安全存储，我们 **不存储** 任何支付信息。

您可以手动充值指定金额的信用，金额必须大于 \$1。此外，您可以启用[自动充值](/docs/support/auto-top-up)，当您的余额低于指定阈值时，系统将自动充值。
